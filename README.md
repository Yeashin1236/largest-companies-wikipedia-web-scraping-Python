# largest-companies-wikipedia-web-scraping-Python

This project contains a simple web scraping script built with Python to extract information about the world's largest companies from Wikipedia. The data is processed and displayed in a clean, structured format using libraries like pandas, requests, and BeautifulSoup.

**Features**
- Scrapes the list of the largest companies from a specified Wikipedia page.
- Extracts key details such as company name, revenue, profit, number of employees, and industry.
- Organizes the data into a pandas DataFrame for easy analysis.
- Handles minor data cleaning (like removing unwanted characters).
- Provides an option to export the data to a CSV file.
